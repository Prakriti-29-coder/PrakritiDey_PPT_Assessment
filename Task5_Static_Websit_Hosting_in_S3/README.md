# Static Website Hosting on AWS S3 (Template Version)

## Objective
Host a static website using an HTML template from HTML5 UP on Amazon S3.

## Steps Taken
1. Downloaded `Photon` HTML template from HTML5 UP.
2. Created S3 bucket named `prakriti-static-template`.
3. Uploaded all template files (`index.html`, `css/`, `images/`, etc.).
4. Enabled static website hosting in bucket properties.
5. Configured bucket policy for public read access.
6. Verified website using the endpoint URL.
