# Task 19: Azure Storage Account (Blob, File, Table)

**Objective:** Create a storage account and explore Blob, File, and Table storage.

---

## Step 1: Create Storage Account
- Login to [Azure Portal](https://portal.azure.com)  
- Search **Storage Accounts → Create**  
- **Resource Group:** ppt-rg  
- **Storage Account Name:** pptstorage123 (unique)  
- **Region:** East Asia  
- Keep default settings → **Review + Create → Create**  

---

## Step 2: Explore Blob Storage
- Go to **Containers → + Container → sample-container**  
- **Upload** a sample file to the container  

---

## Step 3: Explore File Share
- Go to **File Shares → + File share → sample-fileshare → Create**  
- Open file share → Upload a sample file  

---

## Step 4: Explore Table Storage
- Go to **Tables → + Table → sample-table → OK**  
- Add sample entity:
  - **PartitionKey:** Student  
  - **RowKey:** S001  
  - **Name:** Prakriti  
  - **Age:** 21  


---

- **File Share:** sample-fileshare with sample file
