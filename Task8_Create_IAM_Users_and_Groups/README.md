# Task 8: Create IAM Users and Groups

## Objective
Manage users and permissions in AWS using IAM.

## Steps Followed

1. Logged into AWS Management Console.
2. Navigated to IAM service.
3. Created IAM group named "Adminss" and attached "S3 Bucket" policy.
4. Created IAM user "Prakriti" with AWS Management Console access.
5. Added the user to the "Adminss" group.
6. Verified user permissions under the Permissions tab.
